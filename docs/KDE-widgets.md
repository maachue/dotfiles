# KDE Installation

## Widgets config

Panel Colorizer:

```
Load Black

Appearance:
Color: Custom > #11111B
       Alpha: 1.00 (can change)

Shape: Radius > {9;9;9;9}
       Margin > {-2; 0; 0; -2}
```

Digital Clock:

```
Appearance:
Data format: Custom > d MMM yyyy
Infomation: Tick Show date; Always beside time
Text display: Manual > 12pt Work Sans Light
```
